# Prospect Hub — Sales Development Rep

Clay-style prospecting and enrichment platform for Emika AI Employee.

## Features
- **Clay-style enrichment table** with inline editing, bulk actions, color-coded cells
- **ICP Builder** — Define ideal customer profiles with scoring criteria
- **Waterfall enrichment** — Apollo, ContactOut, RocketReach, Hunter.io
- **Campaign builder** — Multi-step email sequences with merge fields
- **Email templates** — Library with cold email frameworks
- **CSV import/export** — Bring your own prospect lists
- **Apollo API integration** — Real prospect search and enrichment

## Quick Start
```bash
npm install
node server.js
# Open http://localhost:3000
```

## Stack
- Express.js backend with JSON file persistence
- Vanilla JS SPA frontend (no framework dependencies)
- DM Sans typography, dark theme (#06060a)
